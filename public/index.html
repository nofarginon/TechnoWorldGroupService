Techno World Group API

This is Techno World Group API
explain how to use on this app.
You can use in 3 methods:
getSongsData, getPlayListByPreferences and getPlayListByProPreferences

Present your songs list:

http verb:get

/getSongsData

If you would like to see all the songs in this app you will have to go into this link:
https://technoworldgroup.herokuapp.com/getSongsData
You will get a response in json view

example of error message by json view


{
 "error": "cannot get songs"
}

Looking for playlist by Techno genre and period:

http verb:get

/getPlayListByPreferences

If you would like to looking for playlist by insert youe favorite genre in Techno and period you will have to go into this link:
https://technoworldgroup.herokuapp.com/getPlayListByPreferences?genre=minimal&period=modern
In this serach we were looking for minimal music from modern time.
If there is a match playlist with this preferences, you will get a response in json view with this playlist
message error in json view otherwise

example of error message by json view


{
 "error": "There is no playlist by these Preferences"
}

Looking for playlist by pro preferences:

http verb:get

/getPlayListByProPreferences

If you would like to looking for a playlist by advanced preferences in our playlist machine you will have to go into this link:
https://technoworldgroup.herokuapp.com/getPlayListByProPreferences?bpm=140&accent=1&bassdrums=5&scatter=3&mfx=max&melodie=exciting&cymbal=3
If there is a playlist with these preferences, you will get a response in json view with all of playlists with these preferences
message error in json view otherwise

example of error message by json view


{
 "error": "There is no playlist by these Preferences"
}

Create Playlist:

http verb:post

/createPlaylist

If you would like to create your own playlist you will have to insert these values:
PlaylistName:
Djname:
country:
about:
pick songs from a list
https://technoworldgroup.herokuapp.com/createPlaylist
You will get a response in json view

example of data to send in raw data


{

 "DjName:newuser"
 "country:israel"
 "about:new playlist best one"

}
